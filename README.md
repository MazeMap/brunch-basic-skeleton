brunch-basic-skeleton
=====================

A basic skeleton for brunch

# Example initialization process:
```
npm install -g brunch

mkdir my-new-app

cd my-new-app

brunch new https://github.com/MazeMap/brunch-basic-skeleton

npm start
```
Voila!
